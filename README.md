Proyecto: Detección de Fraude en Votaciones

Descripción
Sistema para detectar votos fraudulentos en elecciones estudiantiles mediante análisis de IP y patrones de tiempo.

Objetivo
Desarrollar un MVP que identifique y bloquee votos sospechosos en tiempo real.

- Métricas de fraude detectado
